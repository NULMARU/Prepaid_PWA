1. 앱 소개

선입금대장은 음식점에서 단체·직원 식대 선입금을 관리하는 앱입니다. 직원이나 이용자가 미리 입금한 식대 잔액을 기록하고, 식사할 때마다 서명과 함께 차감하는 장부 역할을 합니다.

주요 기능

  • 부서별 직원 잔액 한눈에 확인
  • 식사 사용 시 서명을 받아 기록
  • 일자별·직원별 거래 이력 조회
  • 사용 취소 및 잔액 조정
  • 데이터 백업/복원 (JSON 파일)
  • 장부 엑셀(CSV) 내보내기
  • 4자리 비밀번호 잠금

사용 환경

  • 스마트폰, 태블릿, PC 모든 브라우저에서 사용 가능
  • 인터넷 없이도 동작 (오프라인 지원)
  • 설치 없이 바로 사용 (홈 화면에 추가하면 앱처럼 동작)

2. 초기 설정 (처음 사용 시)

앱을 처음 열면 2단계 초기 설정이 나타납니다.

1단계: 우리 가게 등록

1 음식점 이름, 주소, 전화번호를 직접 입력하고 [입력 완료] 버튼을 누릅니다.
예: 음식점 이름 → "맛있는식당"
2 입력한 가게 정보를 확인하고 [다음]을 누릅니다.
가게 등록을 나중에 하려면 [건너뛰기]를 누르세요. 설정에서 언제든 수정할 수 있습니다.

2단계: 부서 목록 설정

3 기본 그룹이 미리 채워져 있습니다. 필요 없는 항목은 × 버튼으로 삭제하고, 추가할 부서·단체명은 입력 후 [추가]를 누릅니다.
4 부서 목록을 확인하고 [완료]를 누릅니다.

비밀번호 설정

5 초기 설정 완료 후 4자리 숫자 비밀번호를 설정합니다. 같은 번호를 두 번 입력하면 설정됩니다.
비밀번호를 잊으면 PIN만 초기화할 수 없습니다. 5회 실패 후 앱 초기화를 선택하면 모든 로컬 데이터가 삭제됩니다.

3. 홈 화면 — 잔액 확인과 사용 등록

홈 화면은 앱의 메인 화면입니다. 부서별로 직원 목록과 잔액이 표시됩니다.

화면 구성

영역설명
상단가게 이름, 앱 제목, 검색 초기화 버튼
필터부서 선택 드롭다운 + 직원명 검색창
직원 카드이름, 부서, 잔액, [사용] 버튼
하단 요약활성 직원 수와 총 잔액
탭 메뉴홈 / 이력 / 설정 전환

직원 검색

  • 부서 필터: 상단 드롭다운에서 특정 부서만 표시
  • 이름 검색: 검색창에 직원 이름 일부 입력
  • [검색 초기화] 버튼으로 필터 해제

4. 식사 사용 등록 (서명 포함)

직원이 식사할 때 잔액에서 차감하는 핵심 기능입니다.

1 홈 화면에서 해당 직원의 [사용] 버튼을 누릅니다.
2 사용 등록 화면이 열립니다. 직원명, 부서, 현재 잔액이 표시됩니다.
3 사용 금액을 입력합니다. (예: 9000)
4 필요시 비고를 입력합니다. (예: 점심, 저녁, 특식 등)
5 직원에게 서명을 받습니다. (화면 아래 서명란에 손가락이나 펜으로 서명)
6 [서명 후 저장] 버튼을 누르고, 확인 팝업에서 [확인]을 선택합니다.
잔액이 부족하면 사용 등록이 불가합니다. 먼저 충전이 필요합니다.
서명이 잘못되면 [서명 지우기]로 다시 받을 수 있습니다.

5. 이력 화면 — 거래 내역 확인

하단 탭에서 [이력]을 누르면 모든 거래 기록을 확인할 수 있습니다.

보기 방식

탭설명
일자별날짜순으로 모든 거래를 볼 수 있습니다. 하루 사용 합계가 표시됩니다.
직원별직원을 선택해서 해당 직원의 거래만 볼 수 있습니다.

거래 유형

유형의미표시 색상
등록초기 선입금액 등록파란색 (+)
충전추가 입금파란색 (+)
사용식사 차감 (서명 포함)빨간색 (−)
취소사용 취소 (금액 복원)파란색 (+)
조정잔액 수동 조정주황색

사용 취소 방법

1 이력 화면에서 취소할 "사용" 거래의 [취소] 버튼을 누릅니다.
2 취소 사유를 입력하고 확인합니다. 금액이 자동으로 복원됩니다.

6. 설정 화면 — 직원·부서·백업 관리

직원 등록

개별 등록:

1 [설정] → [직원개별등록] 버튼을 누릅니다.
2 부서를 선택하고, 이름과 초기 선입금액을 입력한 뒤 [저장]합니다.

CSV 일괄 등록:

1 [설정] → [직원등록(csv파일)] 버튼을 누릅니다.
2 CSV 파일을 선택합니다. (필수 컬럼: 부서, 이름, 금액 / 헤더 행 필수)
3 미리보기를 확인하고 [임포트 실행]을 누릅니다.
이미 등록된 직원과 같은 이름+부서가 CSV에 있으면 자동으로 추가 충전됩니다.

직원 관리 (수정·충전·숨김)

버튼기능
수정이름, 부서, 메모 변경
충전추가 입금 금액 등록
조정잔액을 수동으로 변경 (사유 필수)
숨김직원을 목록에서 숨김 (데이터 유지, 복원 가능)

부서 관리

  • 설정 화면 하단 "부서 목록 관리"에서 부서 추가/삭제 가능
  • 기관명도 변경할 수 있습니다

비밀번호 변경

  • 설정 → "보안" 항목의 [변경] 버튼을 누릅니다.
  • 새 4자리 비밀번호를 두 번 입력하면 변경됩니다.

7. 데이터 백업과 복원

매일 영업 종료 후 반드시 백업하세요! 기기 고장이나 데이터 손실에 대비합니다.

백업하기

1 [설정] → [💾 데이터 안전 저장] 버튼을 누릅니다.
2 JSON 파일이 다운로드됩니다. 안전한 곳(클라우드, USB 등)에 보관하세요.

복원하기

1 [설정] → [📂 저장한 데이터 불러오기]를 누릅니다.
2 이전에 저장한 JSON 백업 파일을 선택합니다.
3 백업 형식과 데이터 검증 결과를 확인한 뒤 데이터가 복원됩니다.
복원하면 현재 데이터가 백업 파일의 데이터로 완전히 교체됩니다.

장부 엑셀 저장

[📋 장부 엑셀로 저장] 버튼을 누르면 직원별 잔액 현황과 전체 거래 내역이 CSV 파일로 저장됩니다. 엑셀에서 열어 인쇄하거나 보관할 수 있습니다.

8. 자주 묻는 질문 (FAQ)

Q. 비밀번호를 잊었어요.

A. 비밀번호를 5회 틀리면 [앱 초기화] 버튼이 나타납니다. 보안을 위해 PIN만 삭제할 수 없으며, 초기화하면 저장된 데이터(직원, 거래, 서명)가 모두 삭제됩니다.

Q. 직원이 퇴직했어요.

A. 설정에서 해당 직원을 [숨김] 처리하세요. 거래 이력은 보존되고, 홈 화면에서만 사라집니다. 나중에 복원도 가능합니다.

Q. 잔액이 실제와 다릅니다.

A. 설정에서 해당 직원의 [조정] 버튼을 눌러 정확한 잔액을 직접 입력할 수 있습니다. 사유를 입력해야 합니다.

Q. 핸드폰을 바꿨어요. 데이터를 옮길 수 있나요?

A. 이전 기기에서 백업 파일을 저장해두었다면, 새 기기에서 앱을 열고 [저장한 데이터 불러오기]로 복원하면 됩니다.

Q. 인터넷 없이도 되나요?

A. 네. 한 번 접속한 뒤에는 인터넷 없이도 모든 주요 기능이 정상 동작합니다.

Q. 여러 기기에서 동시에 쓸 수 있나요?

A. 이 앱은 각 기기에 독립적으로 데이터를 저장합니다. 여러 기기 동시 사용은 권장하지 않습니다. 한 대의 기기(매장용 태블릿 등)에서 사용하세요.

Q. 전체 초기화는 어떻게 하나요?

A. 설정 → [🔄 전체 초기화]를 누르면 모든 데이터가 삭제됩니다. 초기화 전에 최종 백업이 자동 생성됩니다. 이 작업은 되돌릴 수 없으니 신중하게 진행하세요.
