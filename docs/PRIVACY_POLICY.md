# 개인정보처리방침 (법률 검토 전 초안)

시행 예정 · 운영자: 개인 개발자(NULMARU) · 문의: nulmaru@gmail.com
**이 문서는 법률 전문가 검토 전 초안이며, 실사용 파일럿 개시 전 검토·확정되어야 합니다.**

1. **처리 목적**: 기관 담당자가 암호화하여 전송한 식대 명단을 지정 음식점에 전달(중계)하고, 음식점이 선택한 암호화 장부 백업을 보관하는 것.
2. **처리 항목**: 수신 음식점의 공개키로 암호화된 명단(운영자는 복호화할 수 없음), 총액, 인원수, 무결성 검증용 해시, 인증용 이메일의 해시값(평문 미보관), 접속기록.
3. **보유 기간**: 처리 완료(승인/거절) 후 30일 경과 시 자동 삭제. 암호화 장부 백업은 음식점이 갱신·삭제할 때까지 보관(음식점이 직접 관리).
4. **처리 위탁 및 국외 이전**: 서비스 인프라로 Cloudflare, Inc.(미국)의 Workers/D1을 이용하며, 암호화된 데이터가 국외 서버에 보관될 수 있습니다(이전 국가: 미국 등 Cloudflare 리전 — 확정 리전 확인 후 기재). 이전 항목·기간은 위 2·3항과 같습니다. (개인정보 보호법 제28조의8 제1항 제3호에 따른 공개)
5. **안전성 확보조치**: 종단간 공개키 암호화(운영자 복호화 불가), 전송구간 TLS, 소유 증명 기반 접근통제(챌린지-응답), 처리 완료 데이터 30일 자동 삭제, 접속기록 보관.
6. **정보주체 권리**: 열람·정정·삭제 요구는 명단을 등록한 소속 기관 또는 해당 음식점에 요청하십시오. 운영자는 복호화가 기술적으로 불가능하여 개별 열람을 제공할 수 없으며, 삭제 요청 시 해당 암호문을 삭제합니다.
