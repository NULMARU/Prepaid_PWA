개인정보처리방침

시행일 2026-07-06 · 운영자 널마루 · 문의 nullmaru@naver.com

1처리 목적

기관 담당자가 암호화하여 전송한 식대 명단을 지정 음식점에 중계하고, 음식점이 선택한 암호화 장부 백업을 보관합니다.

2처리 항목

(1) 수신 음식점의 공개키로 암호화된 명단 — 운영자는 복호화할 수 없습니다. (2) 총액·인원수, 무결성 검증용 해시, 인증용 이메일의 해시값(평문 미보관), 접속기록. (3) 음식점이 파일 수신용으로 직접 등록한 연락처(카카오톡 오픈채팅 링크·접수 이메일) — 선택 사항, 언제든 삭제 가능. 오픈채팅 링크는 전화번호·개인 프로필이 노출되지 않는 형식만 허용합니다.

3보유 기간

암호화된 명단은 음식점 수령(승인/거절) 즉시 삭제, 미수령 시에도 최대 3일(72시간) 후 자동 삭제. 요약 정보(총액·인원수·해시)는 30일 보관 후 삭제. 「직접 전달」 방식은 명단이 서버에 일절 저장되지 않음. 암호화 장부 백업은 음식점이 갱신·삭제할 때까지 보관(음식점이 직접 관리).

4처리위탁 및 국외이전

서비스 운영을 위해 Cloudflare, Inc.(미국)의 클라우드 인프라(Workers/D1)에 데이터 처리를 위탁하며, 암호화된 데이터가 국외(미국 등 Cloudflare 리전)에 보관될 수 있습니다. 이전되는 항목·시점·보유기간은 위 2·3항과 같습니다. (개인정보 보호법 §26, §28-8에 따른 공개)

5안전성 확보조치

종단간 공개키 암호화(운영자 복호화 불가), 전송구간 TLS, 소유 증명 기반 접근통제, 수령 즉시 파기 및 미수령분 자동 파기.

6정보주체의 권리

열람·정정·삭제 요구는 명단을 등록한 소속 기관 또는 해당 음식점에 요청하십시오. 운영자는 데이터를 복호화할 수 없어 개별 열람을 제공할 수 없으며, 삭제 요청 시 해당 암호문을 삭제합니다.
