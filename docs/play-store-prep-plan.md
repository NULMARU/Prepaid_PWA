# Play Store 준비 계획

내부 테스트가 끝난 뒤 진행할 항목입니다.

## 앱 패키징

- TWA 기반 Android 프로젝트 생성
- package name 확정
- 앱 서명키 생성 및 백업
- target SDK 요구사항 확인
- 내부 테스트용 AAB 생성

## 스토어 등록 자료

- 앱 이름, 짧은 설명, 긴 설명
- 512px 고해상도 앱 아이콘
- 휴대전화 스크린샷
- 개인정보처리방침 URL
- 지원 이메일
- 앱 카테고리

## 데이터 안전성

- 로컬 저장 데이터 항목 정리: 직원명, 부서명, 선입금/사용 거래, 서명 이미지
- 외부 서버 전송 없음 명시
- 사용자가 직접 백업 파일을 생성하고 보관한다는 점 명시
- Web Speech API 사용 시 음성 인식 처리는 브라우저/플랫폼 정책을 따른다는 점 검토

## 운영 체크리스트

- 내부 테스트 결과 정리
- 복원/백업 실패 시 대응 문서 작성
- 기기 분실 시 데이터 복구 불가 가능성 안내
- 앱 삭제/초기화 방법 안내
- 릴리스 노트 작성
