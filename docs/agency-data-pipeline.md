# 기관·부서 데이터 파이프라인

전국 광역·기초 자치단체와 각 본청의 '과' 목록을 담은 `agency-index.json` /
`agency-depts/{region}.json` 를 생성·갱신하는 절차를 정리한다.

## 출처

- **행정안전부 행정표준코드 관리시스템**(code.go.kr) — "기관코드" 전체자료.
  UTF-8 TSV, 헤더 1행 + 약 46.9만 행(폐지 조직 포함). 2026-07 스냅샷 사용.

## 재생성 절차 (연 1회 권장)

```bash
# 1) code.go.kr 에서 기관코드 전체자료(TSV, UTF-8) 내려받기
# 2) 스크립트 실행 (원자료 경로만 인자로)
node harness/build-agencies.mjs <org0_utf8.txt>
# → agency-index.json, agency-depts/*.json (17개) 재생성
```

스크립트(`harness/build-agencies.mjs`)가 곧 파이프라인 전체다. Node 표준
라이브러리만 사용한다.

## 필터 규칙 (원자료에서 직접 검증해 확정)

- **존속만**: `존폐여부` 열 == `0`. (폐지행은 `1`이며 전부 `폐지일자` 보유 —
  존속행은 폐지일자가 NULL 또는 공란. 46.9만 행 중 존속 약 13.2만.)
- **본청(자치단체)**: `유형분류_대` == `02`(자치행정조직) & `기관코드`==`대표기관코드`
  & `유형분류_중` ∈ {`01` 광역, `02` 기초}.
- **본청 '과'**: `유형분류_대`==`02` & 최하위기관명이 '과'로 끝남 &
  `유형분류_중` ∈ {`01`,`02`,`09`}(광역 자체·기초 자체·보조기관).
  분류체계가 지자체마다 불균일해서(같은 과라도 어떤 곳은 보조기관 09·소05,
  다수 군은 기초 자체 02·소03, 일부 도청은 광역 01로 태깅) 소(小)코드 단일
  기준은 신뢰할 수 없다. 상위 경로에 **읍·면·행정동** 또는
  **소방·보건소·사업소·공단·상수도·농업기술·연구원·의회·직속기관·출장소** 표지가
  있으면 본청 밖 조직으로 보고 제외한다(팀·계·읍면동·보건소·사업소·직속기관·
  의회사무국 제외 원칙).
- **제주 행정시(제주시·서귀포시)**: 하부행정기구라 국 소속 과가 중09·소05로,
  일부는 중03·소06으로 분류되어 특례 처리한다.

> awk 등 로케일 의존 도구에서 한글 문자열 비교가 오작동(예: "동"=="과"가 참)하는
> 환경이 있어, 모든 한글 필터링은 Node(JS)에서 수행한다.

## 2026-07-01 행정구역 개편 반영

원자료는 2026-07-01 시행 개편을 이미 반영한다(웹으로 교차 확인).

- **인천**: 2군 8구 → 2군 9구. 중구·동구→제물포구, 영종구 신설, 서구→서해구 개칭,
  검단구 신설. → 기초 자치구 +1 (아래 226 대조의 +1 원인).
- **광주+전남 통합**: '전남광주통합특별시'(최상위코드 6130000)로 출범.
  그러나 이 앱의 파일 계약은 `gwangju`·`jeonnam` **2개 지역**을 요구하고 앱이 이미
  그 계약으로 구현돼 있으므로, 통합시 산하 기초자치단체를
  **광주(자치구 5) / 전남(시군 22)** 으로 되분리해 17지역 구조를 유지한다.
  통합시의 **광역 본청 과**는 광주·전남으로 쪼갤 수 없어 `광주광역시청`·`전라남도청`의
  부서 목록은 비운다(앱은 부서 미보유 기관에 공통 부서 템플릿을 표시).

## 서울 큐레이션 방침

서울(시청 + 25개 구청)은 자동 추출 대신 기존 `agency-departments.json` 의
수작업 큐레이션 부서를 그대로 이식한다(`source: "큐레이션(2026-06-16)"`, id 재사용).
품질 지표로 광진구청 1곳을 자동 추출과 대조한 결과, 본청 과 32/33 일치(자동은
보건소 과 4건을 규칙대로 제외, 큐레이션이 누락한 평생교육과 1건은 자동이 포착).

## 품질 한계

- 광역시·특례시(예: 경기도청·부산광역시청)는 본부·북부청사·일반구 등 같은
  자치단체 내 다수 조직의 과가 함께 잡혀 목록이 길 수 있다(누락보다 과다 포함).
- '과' 명칭·중분류 태깅이 지자체마다 달라, 드물게 경계 사례가 포함/누락될 수 있다.
- 광주광역시청·전라남도청은 상기 통합 사유로 부서 목록이 비어 있다.
- 원자료 스냅샷 시점 이후의 조직 개편은 재실행 전까지 반영되지 않는다.
