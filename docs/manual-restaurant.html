1설치하기

아래 주소로 접속하세요.

https://app.bapjangbu.com/

꼭 홈 화면에 추가해서 쓰세요! 그래야 앱처럼 편하게 열 수 있어요.

1
크롬으로 위 주소에 접속해요.
2
오른쪽 위 점 3개(⋮) 메뉴를 눌러요.
3
홈 화면에 추가를 눌러요.

아이폰(사파리)에서 추가하기

1
사파리로 위 주소에 접속해요.
2
아래쪽 공유 버튼(⬆️)을 눌러요.
3
홈 화면에 추가를 눌러요.

2처음 설정

앱을 처음 열면 순서대로 화면이 나와요. 모르는 항목은 건너뛰어도 괜찮아요.

1
가게 정보 (1/3)

음식점 이름, 주소(지역), 전화번호를 입력해요.

주소에 "광진구 구의동"처럼 구 이름을 써주면, 담당 구청이 자동으로 선택돼요.

2
부서 목록 설정 (2/3)

기관을 선택해 필요한 부서(과)를 고르거나, 직접 부서명을 입력해 추가할 수 있어요. 나중에 설정에서 언제든 바꿀 수 있어요.

3
연락처 (3/3, 선택·권장)

구청 담당자가 명단 파일을 보낼 곳이에요. 카카오톡 오픈채팅 링크나 접수 이메일을 입력하세요. 지금 몰라도 나중에 하기를 누르면 돼요.

4
비밀번호(PIN) 설정

앱을 열 때 쓸 숫자 4자리를 정해요. 한 번 더 입력해 확인해요.

5
완료 화면

설정이 끝나면 두 갈래 중 하나를 골라 시작할 수 있어요.

  • 직원 등록하기 — 직원과 초기 선입금을 등록해요.
  • 구청 선금 받기 시작 — 우리 가게를 등록해 기관 신청을 받아요.

둘 다 지금 하지 않으려면 나중에 할게요 (홈으로)를 누르세요.

3직원 등록

설정 탭 → 직원 목록 관리에서 직원을 등록해요.

방법 1 — 빠른 등록

직원 목록 관리 화면 아래쪽의 빠른 등록 칸에 부서명 · 직원명 · 선입금액(선택)을 입력하고 추가를 누르면 바로 등록돼요. 부서가 없으면 자동으로 만들어져요.

방법 2 — 명단 파일 등록 (CSV)

명단 파일 등록(CSV) 버튼을 눌러 엑셀에서 저장한 CSV 파일을 불러오면 여러 명을 한 번에 등록할 수 있어요. 이미 등록된 직원은 금액이 추가로 충전돼요.

전화번호 등록 + 문자 안내 동의

각 직원 줄에서 번호 등록을 눌러 전화번호를 넣고, 문자 안내 동의 체크박스를 켜면 사용 등록 후 문자 앱이 자동으로 열려요.

전화번호는 이 폰에만 저장되고, 서버로 전송되지 않아요.

4매일 쓰기

1
홈 화면에서 직원 이름을 누르고, 직원 카드의 사용 버튼을 눌러요.
2
사용 금액을 입력해요.
3
화면에 직원 서명을 받아요.
4
서명 후 저장을 눌러 저장해요.

문자 안내 동의와 전화번호가 등록된 직원은, 저장 직후 문자 앱이 자동으로 열려요. 보내기만 눌러 주면 끝이에요(자동으로 발송되지는 않아요).

직원별 충전 버튼으로 선입금을 추가할 수 있고, 홈 화면 위쪽에서 오늘 사용 금액과 전체 잔액을 바로 확인할 수 있어요.

5구청 선금 받기

설정 탭 맨 위에서 시작하기를 눌러요.

1
가게 이름과 지역으로 검색해요 (예: 지역 광진구 구의동).
2
검색 결과에서 우리 가게를 찾아 이 가게를 눌러 등록해요.

등록이 끝나면, 받는 방법은 두 가지예요.

① 앱으로 바로 받기

① 신청 확인하기를 눌러 새 선금 신청을 확인하고, 내용을 확인한 뒤 등록해요. 받는 즉시 서버에서 삭제되고, 받지 않으면 최대 3일까지만 보관돼요.

② 파일·QR로 받기

담당자가 준 파일을 열거나, ② QR(사각 코드)로 받기로 QR을 스캔해요. 이 방법은 서버를 거치지 않아요.

등록되면 기관·부서·직원·금액이 우리 가게 장부에 자동으로 추가돼요.

6장부 지키기 (중요)

설정 탭의 장부 지키기 카드에서 관리해요.

버튼하는 일
장부 안전 저장엑셀용 CSV와 복원용 데이터 파일을 함께 저장해요. (단, 가게 열쇠는 이 파일에 들어가지 않아요)
내 열쇠 백업 (중요)구청 연동을 쓰신다면 꼭 따로 백업하세요. 암호는 8자 이상으로 정해야 해요.
저장한 데이터 불러오기(복원)안전 저장해둔 데이터 파일(JSON)로 되돌려요.
장부 검사거래 기록에 이상이 없는지 쉬운 말로 확인해요.

📱 폰을 잃어버려도 장부 되찾기

구청 선금 받기를 시작하면 쓸 수 있는 기능이에요.

  • 지금 저장하기 — 내용을 잠근 채로 서버에 보관해요.
  • 새 폰에서 불러오기 — 보관된 내용을 새 기기로 가져와요.
내용은 잠겨 있어서 서버도 내용을 볼 수 없어요.

7비밀번호를 잊었을 때

잠금 화면에서 PIN을 잊으셨나요?를 누르세요.

1
① 백업 파일로 복구 — 저장해둔 백업 파일이 있으면 선택하세요. 데이터를 확인한 뒤 새 비밀번호를 설정해요.
2
백업 파일이 없다면 모든 데이터를 지우고 처음부터 다시 시작할 수 있어요.

8자주 묻는 질문

돈이 드나요?
아니요, 무료예요.
이 앱으로 돈이 오가나요?
아니요. 이 앱은 장부만 기록해요. 돈은 오가지 않아요.
내 자료는 어디에 저장되나요?
이 폰에 저장돼요.
인터넷이 없어도 되나요?
장부 기록(사용·충전)은 인터넷 없이도 돼요. 구청 선금 받기·검색만 인터넷이 필요해요.
폰을 바꾸면 어떻게 하나요?
6번 "장부 지키기"를 참고하세요.
