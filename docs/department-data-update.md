# 기관 부서 목록 업데이트 방식

부서 목록은 앱 코드와 분리된 `agency-departments.json` 파일에서 관리합니다.

## 일반 업데이트

1. 공식 조직도에서 변경된 기관의 과 목록을 확인합니다.
2. `agency-departments.json`에서 해당 기관의 `departments` 배열만 수정합니다.
3. `updatedAt` 값을 수정일로 변경합니다.
4. 정적 사이트에 배포합니다.
5. 앱의 설정 화면에서 `목록 새로고침`을 누릅니다.

스키마가 그대로라면 앱 코드를 새로 개발하거나 빌드할 필요는 없습니다. 정적 배포 환경에서는 JSON 파일만 바꿔 배포하면 됩니다.

## 앱 코드 수정이 필요한 경우

- 기관 데이터 구조가 바뀌는 경우
- `과` 외에 팀, 센터, 담당관 등을 추가로 지원하는 경우
- 자동 업데이트 주기, 변경 알림, 검증 리포트 같은 새 기능을 추가하는 경우

## 데이터 범위

- 포함: 서울시청, 서울시 소속 25개 구청, 이름이 `과`로 끝나는 부서
- 제외: 동주민센터, 담당관, 실, 국, 소, 센터, 단
