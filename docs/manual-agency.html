직원 식대 선입금 명단을 음식점에 안전하게 전달하는 도구입니다. 아래 순서를 따라 진행해 주십시오.

1개요·접속

아래 주소로 브라우저에서 바로 접속하십시오. 별도의 설치는 필요하지 않습니다.

https://agency.bapjangbu.com

본 서비스는 결제·이체 기능이 없는 전달 도구입니다. 선금 이체는 기관에서 음식점 계좌로 직접 진행하시기 바랍니다.

21단계 본인 확인

1
기관 이메일(.go.kr 또는 .korea.kr)을 입력합니다.
2
인증번호 받기를 누릅니다.
3
받은 인증번호 6자리를 입력하고 확인을 누릅니다.
※ 현재 파일럿 기간에는 인증번호가 자동으로 입력됩니다. 정식 운영 시에는 이메일로 발송됩니다.

32단계 소속 기관·음식점

1
시·도를 고른 뒤 소속 기관을 선택합니다. (예: 서울특별시 → 광진구청)
2
소속 기관을 고르면 그 기관 관할로 등록된 음식점 목록이 자동으로 나타납니다. 가나다 순으로 정렬되며, 위쪽 칸에 이름을 입력해 목록을 좁힐 수 있습니다.
3
보낼 음식점을 체크합니다. 여러 곳을 함께 선택할 수 있습니다. (특정 음식점을 뺄 때는 제외)
여기서 고른 소속 기관 이름이 이후 명단에 자동으로 들어갑니다. 소속 기관을 선택하지 않으면 3단계(명단 만들기)로 넘어갈 수 없습니다.
※ 앱을 설치·등록한 음식점만 목록에 나타납니다. 원하는 음식점이 보이지 않으면 목록에 없나요? 이름으로 검색을 펼쳐 이름으로 찾을 수 있습니다. 그래도 없으면 해당 음식점에 앱 설치·(재)등록을 안내해 주십시오. (음식점용 안내문: manual-restaurant.html)

43단계 명단 만들기

1
CSV 양식 다운로드를 눌러 양식을 내려받습니다.
2
엑셀에서 다음 항목(순서 고정)을 작성합니다: 음식점명, 부서명, 직원명, 선금금액, 입금자명
기관명은 적지 않습니다 — 2단계에서 고른 소속 기관이 자동으로 들어갑니다.
3
엑셀에서 그냥 'CSV로 저장'해도 됩니다 — 형식은 자동으로 인식됩니다.
4
작성한 파일을 업로드합니다.

문제가 있으면 표 아래에 원인과 해결 방법이 표시됩니다.

54단계 확인

부서별 합계가 실제 이체하실 금액과 일치하는지 마지막으로 대조하십시오.

65단계 보내기

전송 방법은 두 가지입니다. 잘 모르시겠으면 바로 보내기를 사용하십시오.

방식설명
바로 보내기
추천		 암호화된 명단을 서버를 거쳐 음식점 앱으로 보냅니다. 음식점이 받는 즉시 서버에서 삭제되며, 받지 않으면 최대 3일 후 자동 삭제됩니다.
직접 전달 서버에 저장되지 않습니다. 아래 중 하나를 선택합니다.
  • 파일로 저장 → 이메일 첨부 (권장 1순위)
  • 카톡으로 보내기 (소속 기관 내규를 먼저 확인)
  • QR: 화면 보여주기 · 인쇄 · QR 이미지로 저장 — QR은 사장님 폰이 아닌 다른 화면·종이에 있어야 찍을 수 있습니다.

7개인정보·보안 요약

  • 명단은 브라우저에서 암호화되어 해당 음식점만 열람할 수 있습니다. 서버·운영자는 열람할 수 없습니다.
  • 전화번호는 수집하지 않습니다.
  • 직원 본인의 사전 동의 확보는 기관의 책임입니다. (동의 문구는 전송 시 표시되는 파일럿 배지를 참고하십시오. 법률 검토 후 확정됩니다.)

자세한 내용은 개인정보처리방침(초안)을 확인하십시오.

8보낸 후 할 일

  • 음식점에 전달 사실을 알려 주십시오.
  • 완료 화면을 캡처해 내부 기록으로 보관해 주십시오.
  • 음식점이 3일 안에 받지 않으면 자동으로 삭제되므로, 필요하면 다시 보내 주십시오.
