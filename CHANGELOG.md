# 변경 이력 (CHANGELOG)

이 문서는 선입금대장 PWA의 주요 변경 사항을 기록합니다.

## [2.4.1] - 2026-06-04

### 수정 (Fixed)
- **복원 버그 수정**: 안드로이드 브리지 복원 콜백 `onRestoreData`가 JSON만 파싱하고
  실제 복원을 수행하지 않던 빈 스텁 문제를 해결했습니다. 파일 복원과 동일한 로직을
  `restoreFromParsed`로 공통화하여 두 경로가 일관되게 동작합니다.
- **데이터 보존**: 복원 데이터에 직원/거래가 있으나 백업 메타에 `setupComplete`가
  없을 때, 복원 후 초기 설정 마법사로 되돌아가 데이터에 접근하지 못하던 문제를
  방지하도록 `setupComplete`를 유지합니다.

### 추가 / 개선 (Added / Changed)
- **가게검색 폴백**: 초기 설정의 공공데이터 가게 검색 API 호출이 실패하면(네트워크/오류)
  자동으로 직접 입력 모드로 전환하여 가게 등록을 계속 진행할 수 있습니다.
- **캐시 동기화**: 서비스 워커 캐시 이름을 앱 버전(`prepaid-ledger-v2.4.1`)과
  일치시켜 신버전이 정상 배포되도록 하고, `manual.html`을 캐시 자산에 추가했습니다.
- 앱 버전을 2.4.0에서 2.4.1로 올렸습니다.
